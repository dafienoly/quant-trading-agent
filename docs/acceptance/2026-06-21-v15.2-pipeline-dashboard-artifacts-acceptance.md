## 缺陷与剩余风险

已发现并修复两个 S2 缺陷：

1. fresh runner 缺少 `runtime/` 导致 pytest 21 个 setup errors；已在 pytest 前创建目录。
2. pytest 失败时 regression JSON 仍可能显示 PASS；已聚合五个 step outcome，任一失败强制报告 fail。

当前未发现未修复的 S0-S2 阻断缺陷。

非阻断事项：

- Windows self-hosted runner 的 PowerShell fallback 尚未在本地 Windows 环境执行。
- 当前环境没有 Playwright/Chromium，未做浏览器截图 smoke；HTML 生成和结构由自动化测试覆盖。
- GitHub Actions 给出 Node 20 runtime 弃用警告，属于 action 平台迁移提示。
- 修复提交的 synchronize 事件未登记 attached check；最新 head 已通过 workflow dispatch，仍需用 edited 事件复核 PR check 关联。
