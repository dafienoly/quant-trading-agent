## 4. 关键负向用例

- 缺少 API key：`unavailable`，不创建 client。
- 缺少 SDK：`unavailable`。
- timeout：按配置有限重试，最终 `timeout`。
- 非瞬时错误：不重试，返回 `api_error`。
- asyncio cancellation：向调用者传播，不伪装成功。
- 缓存目录错误：在创建 client 前 `api_error`。
- 空 content、非法 JSON、非 object、未知 schema、字段缺失：`invalid_response`。
- 未知工具、写工具、参数缺失/类型错误、执行失败、轮次超限：`tool_error`。
- 会话密钥、嵌套 token 和 reasoning content：不进入持久化文件。
- `needs_human_review=false`、`requires_approval=false`：schema 拒绝。
- 风控和执行模块提案：BugFixAgent 阻断。
