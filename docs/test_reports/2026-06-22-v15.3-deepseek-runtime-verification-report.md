### 5.2 单元测试

```bash
./.venv/bin/python -m pytest tests/test_model_router.py -q \
  -k 'not llm_status_endpoint' \
  --basetemp=runtime/pytest-tmp-v15-3-model-router-unit
```

结果：`5 passed, 1 deselected`。

被排除用例是 FastAPI 状态接口用例。完整文件首次执行结果为 `5 passed, 1 failed`，初始阻断原因是当前 `.venv` 未安装 FastAPI。

补装 FastAPI、loguru、httpx、pydantic、python-dotenv 和 openai 后复跑一次，完整文件仍为 `5 passed, 1 failed`，但阻断点推进到未安装的 Pandas。BugFixAgent 测试文件也因顶层导入 API app 并间接导入 Pandas 而未能收集。两次均为环境依赖错误，不是测试断言失败。
