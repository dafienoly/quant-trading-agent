# AgentOps Web Foundation

This is the V16.1 frontend foundation for the AgentOps Control Tower.

## Scope

- React
- Vite
- TypeScript
- Readonly AgentOps API client
- `/agent-pipeline` page skeleton

## Commands

```bash
cd apps/web
npm install
npm run build
npm run test
```

## Notes

This package is a frontend foundation slice. It does not replace the existing Streamlit dashboard and does not implement trading, order, broker, or workflow write operations.
